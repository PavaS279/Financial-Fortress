# **App Name**: Financial Fortress

## Core Features:

- UPI Scam Detector: Analyzes UPI transactions for fraud risk indicators using a proprietary tool and alerts the user to potentially malicious activity.
- Loan Analyzer: Evaluates loan offers and generates a risk assessment report based on the entered financial information.
- Insurance Decoder: Simplifies complex insurance policies into plain language. Generates a user-friendly summary that highlights key terms, exclusions, and benefits.
- Dashboard: Displays key metrics, fraud risk scores, and insights derived from analysis, all within an easy to read and customizable UI.
- Settings: Enables user customization: manages connected accounts, notification preferences and display preferences such as light/dark mode.

## Style Guidelines:

- Primary color: Strong, secure blue (#2962FF) to convey trust and reliability.
- Background color: Light gray (#F0F2F5) to provide a clean, unobtrusive backdrop. Offers a high contrast when in light mode.
- Accent color: Analogous light blue (#3DD5FF) for interactive elements and highlights, to complement the primary and reinforce the secure theme.
- Font pairing: 'Space Grotesk' for headlines and 'Inter' for body text; Space Grotesk is sans-serif and provides a computerized feel, while Inter is a grotesque-style sans-serif with a neutral look.
- Use a set of consistent and clear icons from a single library (e.g., FontAwesome or Material Icons).
- Responsive grid layout adapts to various screen sizes.
- Use subtle animations for transitions and hovers to enhance UX.
